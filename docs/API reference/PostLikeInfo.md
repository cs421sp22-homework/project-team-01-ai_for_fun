# Getpost

Returns all posts related to a user

**URL** : `/postlikeinfo/:user_id`

**Method** : `get`

**Auth required** : No
**Data constraints**

empty

**Data example**

empty


## Success Response

**Code** : `200 OK`

**Content example**

```json
[
    {
        "liked_sum": 2
    }
]
```
**Code** : `500 Internal Server Error`
