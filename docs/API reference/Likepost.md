# Likepost

give a like for a post

**URL** : `/likepost`

**Method** : `POST`

**Auth required** : No
**Data constraints**

```json
{"post_id":"[post_id]"
}
```

**Data example**

```json
{"post_id":"001"
}
```


## Success Response

**Code** : `200 OK`

## Error Response

**Code** : `500 Internal Server Error`
